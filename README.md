# Helperym

Desktop-приложение с модульной архитектурой (Block MM, PhotoSorter, SearchInfo, PurgeAcad и др.).

**Исходный код** — в приватном репозитории разработки. Здесь публикуются только **сборки**.

## Скачать

Перейдите в [**Releases**](https://github.com/LarixRus/Helperym/releases/latest) и скачайте:

| Файл | Назначение |
|------|------------|
| **Helperym.exe** | Ядро приложения (главное окно) |
| **block_mm.zip**, **search_info.zip**, … | Модули (опционально) |

### Установка

1. Создайте папку, например `C:\Helperym\`.
2. Положите туда **Helperym.exe**.
3. Распакуйте нужные zip-модули **в ту же папку** (в архиве уже есть подпапка `block_mm\`, `photo_sorter\` и т.д.).
4. Запустите **Helperym.exe**.
5. Либо установите модуль через кнопку **«Установить модуль из zip…»** в главном окне.

### Зависимости модулей

Ядро собрано со всеми библиотеками, объявленными в `module.json` модулей на момент сборки (например Pillow для PhotoSorter). Новые модули с другими зависимостями могут потребовать обновления ядра.

## Требования

- Windows (основная платформа)
- Python не нужен — всё включено в exe
